CSS 定位布局

定位布局可以让元素脱离文档流，钉在你指定的位置。

相对定位

移动元素的位置，并且保留原本的位置。

参照元素原本的位置进行移动。

div {
	position: relative;
}

绝对定位

移动元素的位置，并且去除原本的位置。

默认参照整个页面的位置进行移动。如果父级元素有定位属性，则参照父级元素的位置进行移动。

div {
	position: relative;
}

div img {
	position: absolute;
}
  • 如果子元素的父元素没有定位属性，那么子元素的定位将参照离自己最近的拥有定位属性的上级元素。
  • 有滚动条时，元素会随着内容滚动。

粘性定位

元素可以在页面滚动时，钉在指定位置。

div {
	position: sticky;
}

固定定位

元素可以固定到页面的指定位置。

参照整个页面指定位置。

div {
	position: fixed;
}
